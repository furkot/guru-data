# @furkot/guru-data

Data used by Furkot modules dealing with GPX files with data specific to [Guru] offline maps and navigation app.

## Install

```sh
$ npm install --save @furkot/guru-data
```

## Usage

```js
const { toGuru, toFurkot } = require('@furkot/guru-data');
```
